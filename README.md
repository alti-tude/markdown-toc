# TOC GENERATOR

## Usage
1. compile and run
    ```bash
    g++ toc_gen.cpp -o toc_gen
    ./toc_gen
    ```
2. Enter the file path(relative to the current directory).  
3. A file called "withTOC.md" will be generatod in the directory containing the input file.
